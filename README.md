# Customer Churn Risk Analysis

Analyzed 2,800-row dataset of unique customers exhibiing overall churn rate of 57%. Extracted key drivers of churn and provided actionable insights for businesses.

## Files
- 'tbd' - Tableau storyboard visualizing insights
- 'tbd' - Jupyter notebook with logistic regression coefficient chart
- 'tbd' - Kaggle dataset

## Key Insights
- Usage, recency, payment failures, and support tickets are key drivers of churn
- High-risk customers are 3x more likely to churn than low-risk customers
